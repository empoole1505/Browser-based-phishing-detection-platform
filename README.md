Welly extension contains all files contained in the browser extension

wellyai.py is used to train the machine learning model and is used to output the teained model to pkl file

project_svm.py is used to capture a screen, display gui and use scanned data for input in the model within the pkl file

com.welly.boot should be placed within google-chrome's native messaging configuration folder

bridge.sh is required shellscript which is used to authorise and run the project_svm.py file

project_svm.py IS USED TO TRAIN A RANDOM FOREST MACHINE LEARNING MODEL AND NOT A SUPPORT VECTOR MACHINE DO NOT BE CONFUSED BY THE 'svm' IN THE FILE NAME
